## 在Analysis Workspace中为自动分配活动设置A4T报表 {#a4t-auto-allocate}

自动分配活动可在两个或更多体验中标识一个入选者，并在测试继续运行和学习期间，自动为入选者重新分配更多流量。 自动分配的[!DNL Target] (A4T)的[!DNL Analytics]集成允许您在Adobe [!DNL Analytics]中查看报表数据，甚至可以针对[!DNL Analytics]中定义的自定义事件或量度进行优化。

## 在Analysis Workspace中为自动[!DNL Target]活动设置A4T报表 {#a4t-auto-target}

自动生成[!DNL Target]活动的[!DNL Target] (A4T)集成的[!DNL Analytics]使用Adobe [!DNL Target]集成机器学习(ML)算法，根据每位访客的个人资料、行为和上下文为其选择最佳体验，同时使用Adobe [!DNL Analytics]目标量度。

尽管Adobe [!DNL Analytics] Analysis Workspace提供了丰富的分析功能，但由于实验活动（手动A/B测试和自动分配）和个性化活动（自动[!DNL Target]）之间存在差异，因此需要对[!DNL Target]面板的默认[!DNL Analytics]进行一些修改才能正确解释自动[!DNL Target]活动。
